## [The Jumping Jack](https://mondal10.github.io/jumping-jack-/)
**Instruction:**
- Spacebar to jump.
- Left arrow key(<-) to move Left.
- Right arrow key(->) to move Right.
- Collect the Stars.
- Dodge the Bombs.
